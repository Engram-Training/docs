---
title: "List Media"
description: "GET /v1/media — List media files with optional category filtering"
---

## Endpoint

```
GET /v1/media
Authorization: Bearer sk_live_...
```

## Query Parameters

| Param | Type | Default | Description |
|---|---|---|---|
| `category` | string | | Filter: `photo`, `video`, `audio`, `document` |
| `limit` | number | `50` | Max results (capped at 100) |
| `offset` | number | `0` | Pagination offset |

## Example

```bash
# List all media
curl "$API/v1/media" -H "Authorization: Bearer $API_KEY"

# List only photos
curl "$API/v1/media?category=photo&limit=20" -H "Authorization: Bearer $API_KEY"
```
