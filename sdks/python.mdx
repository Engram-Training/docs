## Quick Start

```python
from engram import Engram

client = Engram(
    api_key="sk_live_your_key_here",
    base_url="https://api.engram.training/v1"
)

# Store a memory
memory = client.store(
    type="semantic",
    key="facts/user-name",
    content="User name is Alice",
    importance=0.8,
    expires_in="30d",
    metadata={"tags": ["user"], "visibility": "private"}
)

# Retrieve
result = client.get(memory.id)
print(result.content)

# Search by content
results = client.search(content_query="Alice", type="semantic")

# Update
client.update(memory.id, content="User name is Alice Smith")

# Delete
client.delete(memory.id)
```

## File Uploads

```python
# Upload
with open("report.pdf", "rb") as f:
    file = client.upload(
        file=f,
        type="file",
        key="docs/quarterly-report",
        mime_type="application/pdf"
    )

# Download
data = client.download(file.id)
with open("downloaded.pdf", "wb") as f:
    f.write(data)
```

## Vector Search

```python
similar = client.vector_search(
    embedding=[0.85, 0.15, 0.25, 0.1, 0.75],
    threshold=0.5,
    limit=5
)

for result in similar:
    print(f"{result.key}: {result.score:.3f}")
```

## Async Support

```python
from engram import AsyncEngram

client = AsyncEngram(api_key="sk_live_...")

memory = await client.store(
    type="semantic",
    key="facts/async-test",
    content="Async works!"
)
```

<Note>
  The Python SDK is currently in development. For now, use the REST API directly with `requests` or `httpx`. The examples above show the planned API surface.
</Note>
