## What is Shelby?

[Shelby Protocol](https://shelby.xyz) is a decentralized storage network built on the **Aptos blockchain**. It provides:

- **Blob storage** — Content-addressable, tamper-proof data storage
- **On-chain coordination** — Move smart contracts manage blob lifecycle
- **ShelbyUSD** — Storage token for blob operations

## How Engram Uses Shelby

Every memory's **content** is stored as a Shelby blob on-chain. The blob is identified by a name like:

```
memory/semantic/facts/project-alpha/v1
```

### Blob Naming Convention

```
memory/{type}/{key}/{blobFilename}_v{version}{extension}
```

Examples:
- `memory/semantic/facts/project/v1` — text memory
- `memory/file/docs/report/quarterly_report_v2.pdf` — named file
- `memory/file/test/data/encrypted_v1.enc` — encrypted upload

### Blob Operations

| Operation | Shelby Method |
|---|---|
| Store | `coordination.writeBlob()` |
| Read | `coordination.readBlob()` |
| Delete | `coordination.deleteBlob()` |

## Token Economics

| Token | Purpose |
|---|---|
| **APT** | Gas fees for Aptos transactions |
| **ShelbyUSD** | Storage costs on Shelby network |

Agent wallets are funded with both tokens to cover storage and gas costs.

## Network Configuration

Engram supports both Shelby networks:

| Network | Use Case |
|---|---|
| `shelbynet` | Shelby's own network |
| `testnet` | Aptos testnet (development) |

Configure via `SHELBY_NETWORK` environment variable.
